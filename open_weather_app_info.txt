open weather app

제공해주는 API
1. Current Weather Data (현재 날씨 정보)
- 200,000 개 이상의 도시를 포함한 모든 위치의 현재 날씨 데이터에 액세스
- 전 세계 및 지역 기상 모델, 위성, 레이더 및 방대한 기상 관측소 
  네트워크와 같은 다양한 소스에서 기상 데이터를 수집하고 처리.
- JSON, XML 및 HTML 형식
- 무료 및 유료 구독에 모두 포함.

2. Hourly Forecast 4 days (시간별 4일 일기 예보)
- 4 일 동안 시간별 예보가 제공됩니다.
- 96 개의 타임 스탬프에 대한 날씨 데이터 예측
- JSON 및 XML 형식
- Developer, Professional 및 Enterprise 구독 계획에 포함

3. One Call API 
- 하나의 API 호출을 수행하고 현재, 예측 및 과거 날씨 데이터를 가져옵니다.
- 1 시간 분 예보
- 48 시간 동안의 시간별 예보
- 7 일간의 일일 예보
- 지난 5 일 동안의 과거 데이터
- 전국 기상 경보
- JSON 형식
- 무료 및 유료 구독에 모두 포함
